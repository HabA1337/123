Правила сдачи заданий:

1. Архив формата zip
2. Архив должен называться логин.zip
3
4. Если письмо пришло не вовремя (или не пришло совсем), архив имеет неверное имя или/и формат, или отправлен всем, то задачи считаются не сданными
5. Внутри архива должны содержаться исходные файлы для всех заданных задач (cpp, h, Makefile). Если файлов не хватает (например, нет Makefile), или есть лишние (например, exe, out), то задачи считаются не сданными
6. Файлы должны компилироваться без предупреждений и ошибок, иначе задачи считаются не сданными
7. В результате запуска команды make должен получиться 1 исполняемый файл с именем a.out, содержащий решения задач, заданных на дом; иначе задачи считаются не сданными
8. При использовании чужого кода задачи считаются не сданными
9. Исполняемый файл a.out должен пройти все тесты. Тогда соответствующая задача считается сданной
